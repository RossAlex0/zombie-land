## 🏛️ Architecture :   
  - **SPA** : Expérience fluide, meilleure réactivité

  - **Monolithe full-stack Next.js** : Front et back dans une seule application (API Routes de Next), codebase unifié, types partagés, déploiement simplifié
  
  - **API REST** : Standard universel, simplicité, cacheable

  - **MVC** : Séparation des responsabilités, code maintenable et testable, structure claire pour faire évoluer le projet

  - **Base de données relationnelle** : Données fortement structurées et interconnectées

  - **Containerisation Docker** : Environnement reproductible entre dev, staging et production, isolation des composants
